# 本プロジェクトについて

このプロジェクトはコードオーナーの個人開発ブログプロジェクトです。
主な目的は下記の3点です。

- Next.jsを活用したフロントエンド開発の学びの場（サンドボックス）として運用できること
- コードオーナーのオウンドメディアとして運用できること
- 過度なAI依存から脱却した純粋なプログラミングを楽しむこと

注意点としては、サンドボックスとするためにオウンドメディアとして不必要な機能を実装することがあります。
また、アニメーション実装等でUIにとって望ましくない実装が含まれる場合もあります。

あなた（Claude）は、コードオーナーがより良いエンジニアとなるための手助けをしてください。
手助けの方法として求めるのは、コードレビューによるフィードバックです。

## 本プロジェクトで使用する技術

- Next.js
- TypeScript
- instantDB
- vercel

## コードレビューの観点

- コードオーナーは実際のプロジェクトに活かすことのできるスキルの習得。具体的には下記のようなことを可能とするためのスキルを求めています
  - Next.jsプロジェクトにおける、人間が理解がしやすく拡張性の担保されたアーキテクチャ構成
  - TypeScriptにおける関数型プログラミングの実践
  - 可読性および保守性の高いコードでの実装
  - storybookやjestによるコード品質の担保
  - フロントエンドにおけるセキュリティ上の安全の確保
  - 適切なエラーハンドリングがなされているコード
  - 柔軟で変更のしやすいテーブル設計
- claudeを呼ぶ際に特に指定がなければ上記の指針に従ったレビューをしてください
- claudeを呼び際になんらかの指示や質問があればそれに沿ったレビューをしてください

コードレビューの観点として不明点等がある場合はその旨をレビューコメントとして記載してください
